# Getting Started with WP Security Pilot

This guide will help you get started with WP Security Pilot.

## Installation

1.  Download the latest release from the [GitHub repository](https://github.com/jhd3197/WP-Security-Pilot/releases).
2.  Upload the plugin to your WordPress site via the "Plugins > Add New" page.
3.  Activate the plugin.

## Configuration

After activating the plugin, you can configure it by navigating to "Security Pilot" in the WordPress admin menu.
