# WP-CLI Command Reference

This document lists all of the WP-CLI commands available in WP Security Pilot.

## General Commands

-   `wp security-pilot status`: Displays the current status of the plugin.
-   `wp security-pilot settings get <setting>`: Gets the value of a specific setting.
-   `wp security-pilot settings set <setting> <value>`: Sets the value of a specific setting.

## Firewall Commands

-   `wp security-pilot firewall list`: Lists all firewall rules.
-   `wp security-pilot firewall add <rule>`: Adds a new firewall rule.
-   `wp security-pilot firewall delete <rule>`: Deletes a firewall rule.
